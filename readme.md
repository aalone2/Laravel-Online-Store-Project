## Installation
- DOWNLOAD THE PROJECT
- MOVE THE PROJECT FOLDER TO THE SERVER'S FOLDER
- MAKE A DATABASE NAMED db_final_proj
- IN THE TERMINAL:
    - Run ```composer install```
    - Run ```php artisan key:generate```
    - Run ```php artisan migrate```
